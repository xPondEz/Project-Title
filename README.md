วิธีการติดตั้งและใช้งาน (Installation & Usage)
เนื่องจากโปรเจกต์นี้เป็นไฟล์ HTML แบบ Standalone คุณไม่จำเป็นต้องติดตั้ง Server หรือโปรแกรมเพิ่มเติม

ดาวน์โหลดไฟล์: บันทึกโค้ดเป็นไฟล์ชื่อ bmiweb.html หรือ index.html

เปิดใช้งาน: ดับเบิลคลิกที่ไฟล์เพื่อเปิดผ่าน Web Browser (เช่น Google Chrome, Edge, Firefox, Safari) ได้ทันที

การเชื่อมต่ออินเทอร์เน็ต: จำเป็นต้องเชื่อมต่ออินเทอร์เน็ตเพื่อโหลด Library Chart.js จาก CDN (สำหรับการแสดงกราฟ)
