#  **คู่มือการใช้งานเว็บไซต์ BMI Calculator**

เว็บไซต์นี้เป็นระบบคำนวณค่า BMI พร้อมบันทึกประวัติการคำนวณ โดยผู้ใช้สามารถกรอกข้อมูลส่วนตัว น้ำหนัก ส่วนสูง และดูผลลัพธ์สุขภาพของตนเองได้ รวมถึงย้อนดูประวัติย้อนหลัง

#  **1\. เริ่มต้นใช้งาน**

เมื่อเปิดหน้าเว็บของ BMI Calculator จะพบหน้าแรกซึ่งประกอบด้วย 3 ส่วนหลัก:

1. **แบบฟอร์มคำนวณ BMI (Calculator Form)**  
2. **ผลลัพธ์ BMI (BMI Result)**  
3. **ประวัติการคำนวณ (BMI History)** – จะปรากฏเมื่อคำนวณอย่างน้อย 1 ครั้ง

#  **2\. การคำนวณค่า BMI**

##  **ขั้นตอนการใช้งาน**

1. ไปที่ส่วน **“Calculator”**  
2. กรอกข้อมูลดังนี้:  
   * ชื่อ (Name)  
   * อายุ (Age)  
   * เพศ (Gender)  
   * น้ำหนัก (Weight)  
   * ส่วนสูง (Height)  
   * เลือกหน่วย (Metric หรือ Imperial)  
3. กดปุ่ม **“Calculate BMI”**

# **3\. ดูผลลัพธ์ BMI**

หลังจากกดคำนวณ จะมีข้อมูลแสดงด้านขวา:

* ค่า BMI (เช่น 22.78)  
* หมวดหมู่สุขภาพ (เช่น Normal, Overweight)  
* สีแสดงสถานะ เช่น  
  * ฟ้า \= น้ำหนักน้อย  
  * เขียว \= ปกติ  
  * เหลือง \= น้ำหนักเกิน  
  * แดง \= อ้วน

หมวดหมู่ประเมินตามมาตรฐานสากล เช่น

* \< 18.5 \= Underweight  
* 18.5 – 24.9 \= Normal  
* 25 – 29.9 \= Overweight  
* ≥ 30 \= Obese

# **4\. ประวัติการคำนวณ (BMI History)**

ทุกครั้งที่ผู้ใช้คำนวณ ระบบจะบันทึกข้อมูลลง **LocalStorage อัตโนมัติ**

ประวัติประกอบด้วย:

* วันเวลา (Date)  
* ชื่อ  
* เพศ  
* ส่วนสูง / น้ำหนัก  
* ค่า BMI  
* หมวดหมู่สุขภาพ

## **ฟังก์ชันที่มีในหน้า History**

* **ลบรายการหนึ่งรายการ**  
  ผู้ใช้สามารถกดปุ่มลบ (Delete) เพื่อเอารายการออก  
* **ล้างประวัติทั้งหมด**  
  จะลบข้อมูลจากเว็บ \+ LocalStorage

#  **5\. หน่วยการวัด (Metric/Imperial)**

Metric (ค่าเริ่มต้น)

* น้ำหนัก \= กิโลกรัม (kg)  
* ส่วนสูง \= เมตร (m)

 Imperial

* น้ำหนัก \= ปอนด์ (lbs)  
* ส่วนสูง \= นิ้ว (inch)

ระบบจะคำนวณให้ถูกต้องโดยอัตโนมัติ

#  **6\. การจัดเก็บข้อมูล**

* ข้อมูลทั้งหมดเก็บใน **LocalStorage**  
* ปลอดภัยเพราะไม่ส่งออกภายนอก  
* ปิดเว็บ/ปิดเครื่อง ประวัติยังอยู่เหมือนเดิม

#  **7\. คำแนะนำด้านสุขภาพ (Understanding BMI Section)**

ด้านล่างสุดของเว็บจะมีส่วนอธิบาย:

* ค่า BMI หมายถึงอะไร  
* ช่วงไหนถือว่าปกติ  
* ข้อจำกัดของการใช้ BMI  
  เช่น ไม่เหมาะกับนักกีฬา, ผู้สูงอายุ เป็นต้น

#  **8\. ข้อควรระวัง**

* BMI เป็นการวัดแบบคร่าว ๆ  
* ไม่ควรใช้บ่งชี้สุขภาพแบบ 100%  
* ควรประกอบกับการวัดอื่น เช่น กล้ามเนื้อ ไขมันในร่างกาย

#  **9\. ปัญหาที่พบบ่อย**

| อาการ | สาเหตุ | วิธีแก้ |
| :---- | :---- | :---- |
| คำนวณไม่ได้ | มีช่องว่างไม่ได้กรอก | กรอกให้ครบ |
| ค่าผิดปกติ | ใส่หน่วยผิด เช่น สูง 170 แต่ใส่ 170 เมตร | ใส่ 1.70 |
| History ไม่ขึ้น | LocalStorage ปิด | เปิดใน Browser ปกติ |
| ลบแล้วไม่หาย | ใช้โหมด Incognito | เปลี่ยนเป็นโหมดปกติ |
